== ML and Differential Privacy

=== MIA attack

Membership Inference Attack

Given a training set and a model

* train model
* expose the trained model

Attack pipleine:

. attacker knows the structure of the training dataset structure
* i.e. the features (easy to know just by using the model)
. attacker creats a fake training set
* random generation of feature values given known features
* inference of data distribution to generate features
* if attacker can access a leak they can live large
** they can infer a more accurate distribution of data
. synthetic dataset is unlabeled
* take model and inject synthetic dataset
* see how the model behaves
. train multiple models while splitting the synthetic data into different train/test splits 
* get probability vector for each predicted class
. training set is rebuilt

This works on poorly generalized models

* exploits predicted probability peaks
